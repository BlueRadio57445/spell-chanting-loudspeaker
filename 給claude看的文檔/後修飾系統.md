# 後修飾系統

## 後修飾是什麼

後修飾（PostModifier）是**掛在投射物節點上的子節點**，負責賦予投射物額外的行為。

每個後修飾是一個繼承 `PostModifier extends Node` 的腳本，在 `_ready()` 時連接父節點（投射物）的 signal，或覆寫行為邏輯。

```
ProjectileBase（Area2D）
└── QuadShotPostModifier（PostModifier）
└── [其他後修飾...]
```

## 後修飾符文的職責

後修飾符文（Spell→Spell）的工作是：

1. 接收 SpellList（`Array[Dictionary]`）
2. 遍歷每個 spell，對其 `node`（投射物）呼叫 `add_child(PostModifier實例)`
3. 設定好後修飾的參數
4. 原封不動回傳 SpellList

**行為邏輯屬於 PostModifier 節點本身，不屬於符文。**

## 檔案結構

```
scripts/projectiles/
├── post_modifier.gd                        ← 基類，定義 clone() 介面
└── post_modifiers/
    └── quad_shot_post_modifier.gd          ← 命中時四射
```

## clone() 方法

MultiShot 複製子彈時，需要把已掛上的 PostModifier 也複製到新子彈上。每個 PostModifier 子類必須覆寫 `clone()` 方法，明確複製自身的所有屬性並回傳新實例。

```gdscript
func clone() -> PostModifier:
    var c := QuadShotPostModifier.new()
    c.spawn_scene = spawn_scene
    # ...其他屬性
    return c
```
