# 投射物系統

## 架構

投射物 = **基類場景** + **可插拔的移動模組**

## 檔案結構

```
scripts/projectiles/
├── projectile_base.gd        ← 投射物基類（Area2D）
├── movement_module.gd        ← 移動模組基類（Node）
└── movements/
    ├── linear_movement.gd     ← 直線飛行（預設）
    ├── stationary_movement.gd ← 靜止不動
    ├── orbit_movement.gd      ← 環繞玩家
    ├── homing_movement.gd     ← 追蹤敵人
    └── falling_movement.gd    ← 從天降落（有 BUG，暫停使用）

scenes/projectiles/
└── projectile_base.tscn       ← 基類場景（Area2D + Sprite2D + CircleShape2D r=8）
```

## ProjectileBase（projectile_base.gd）

繼承 `Area2D`，class_name `ProjectileBase`。

### 屬性

| 屬性 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `speed` | float | 300.0 | 飛行速度 |
| `damage` | float | 10.0 | 傷害 |
| `size_scale` | float | 1.0 | 大小倍率 |
| `lifetime` | float | 5.0 | 存活秒數，到期自動 queue_free |
| `effect` | String | "None" | 命中效果名稱（由戰鬥系統組員新增） |
| `effect_time` | float | 0.0 | 效果持續時間 |
| `direction` | Vector2 | RIGHT | 飛行方向 |
| `owner_node` | Node2D | null | 投射物主人（避免自傷） |
| `movement_module` | MovementModule | null | 當前移動模組 |

### 方法

- `setup(owner, direction, speed, damage, effect, effect_time)` — 初始化投射物參數
- `set_movement_module(module)` — 替換移動模組，可在加入場景樹前呼叫
- `_on_body_entered(body)` — 碰撞處理：扣血 + apply_effect + queue_free

### 行為

- `_ready` 時自動尋找子節點中的 MovementModule，找不到則預設 LinearMovement
- `_physics_process` 每幀呼叫 `movement_module.move()`
- Sprite2D 根據 direction 旋轉（`direction.angle() - PI`）

## MovementModule（movement_module.gd）

繼承 `Node`，class_name `MovementModule`。

### 介面

- `init(projectile)` — 投射物 _ready 時呼叫，用於初始化（此時已在場景樹中）
- `move(projectile, delta)` — 每幀由投射物呼叫

## 移動模組清單

### LinearMovement（直線）
- 沿 `direction × speed` 直線飛行
- 無額外參數

### StationaryMovement（靜止）
- 不移動，停在出生位置
- 無額外參數

### OrbitMovement（環繞）
- 繞 owner_node 旋轉
- 參數：`orbit_radius`（預設 80）、`orbit_speed`（預設 3.0）
- 以出生時相對 owner 的角度為起始角

### HomingMovement（追蹤）
- 逐漸轉向目標飛行
- 參數：`turn_speed`（預設 5.0）
- 目標來源優先順序：
  1. 外部指定（`set_target(enemy)`，對應符文系統 Target port）
  2. 自動搜尋最近敵人（group `"Enemy"`）
  3. 都找不到 → 退化成直線
- 指定目標死亡後也退化成直線，不重新搜尋

### FallingMovement（天罰）
- 從高處降落到指定位置
- 參數：`fall_height`（預設 300）、`fall_duration`（預設 0.5s）
- 有 signal `on_landed`
- **目前有 BUG，暫停使用**

## 全域存取

`Main` 有 `static var Instance: Main`，可透過 `Main.Instance` 存取：
- `Main.Instance.player` — 玩家節點
- `Main.Instance.world` — 遊戲世界節點（投射物應 add_child 到此）
- `Main.Instance.projectile_scene` — 投射物基類場景

## Form 系統

效果符文可接收 Form（可選輸入），由前修飾符文提供。Form 是 Dictionary，結構：

```gdscript
{ "movement": "homing", "size_scale": 2.0 }
```

| Key | 值 | 說明 |
|-----|-----|------|
| `movement` | `"bullet"` / `"homing"` / `"orbit"` / `"ground"` | 移動方式，互斥，後蓋前 |
| `size_scale` | float | 大小倍率，可疊加 |

`ProjectileBase.apply_form(form)` 讀取 dict 並套用移動模組與形體變化。

## 投射物變體

| 場景 | 腳本 | 效果 | 特殊 |
|------|------|------|------|
| `fireball.tscn` | ProjectileBase | burn | — |
| `energy_ball.tscn` | ProjectileBase | None | 基礎彈 |
| `ice_ball.tscn` | ProjectileBase | slow | — |
| `poison_ball.tscn` | PenetratingProjectile | poison | 穿透不消失 |

`PenetratingProjectile`（繼承 ProjectileBase）：覆寫 `_on_body_entered`，命中不 `queue_free`，同一敵人只打一次。

## 測試按鍵（暫時）

| 鍵 | 效果 |
|----|------|
| A | 直線子彈，朝滑鼠方向 |
| S | 靜止子彈，出現在滑鼠位置 |
| D | 環繞子彈，繞玩家旋轉 |
| F | 追蹤子彈，追最近敵人 |
